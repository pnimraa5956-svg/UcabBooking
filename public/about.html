About UCab

UCab is an online cab booking application developed using HTML, CSS, JavaScript and Express.js. It allows users to register, login, book rides, calculate fares, view ride history and manage their profile. This project demonstrates a simple cab booking system without using a database.

Mission

To provide safe, reliable and affordable transportation.

Vision

To make travelling easier through smart technology.
