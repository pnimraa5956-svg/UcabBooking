# UcabBooking 🚕 

A full-stack cab booking application built using Node.js, Express.js and MongoDB.

## Features
- User-friendly cab booking system
- Responsive design
- Ride management
- Backend API integration

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## How to Run

Install dependencies:

npm install

Run server:

node server.js
